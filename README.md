# Gestionnaire de contact

## Utilisation
Remplir le formulaire avec le nom et/ou le prénom du contact ainsi que l'adresse mail puis cliquer sur "Ajouter".

## Fonctionnalités

### Suppression
- Cliquer sur supprimer supprimera défénitivement le contact. (sans demande de confirmation)

### Modification
- Cliquer sur modifier permettra de changer soit le nom, soit le mail soit les deux.

## Piste d'améliorations
- Ajouter une fonctionnalité pour qu'une adresse mail déjà enregistrée et identique ne puisse pas être ajoutée.
- Demande de confirmation avant suppression du contact.
- Avoir un design plus travaillé lorsque l'on clique sur "Modifier"
